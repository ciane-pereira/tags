HTML

(voltar)

o HTML é uma linguagem de marcação de hipertexto(HTML) e tem como principal função, marcar e definir a estrutura de uma página na web.

O html é composto de 3 partes, o próprio html, onde estão inseridas dentro dele duas tags a head e body, que são tags com abertura e fechamento.

A tag head é que recebe o título (title) e tem as informações pré-carregadas ao acessar a página da internet pelo navegador.

Já a tag body contém as informações visualizadas pelos usuários.

Principais Tags

Há várias tags disponíveis para compor o seu código, mas aqui serão apresentadas algumas, que digamos, são as principais.

Atributos

(voltar)

Os atributos são usados para personalizar as tags, modificando sua estrutura e funcionalidade. Os atributos são utilizados para atribuir uma classe ou id a um elemento.

A maioria das tags tem seus próprios atributos, mas há atributos que são genéricos e utilizados em tags HTML, veja alguns tipos de atributos:

  • class: Atribui uma classe pode ser utilizada para um ou mais elementos
  • id: Atribui um id ao elemento que é único
  • lang: Define o idioma principal do elemento
  • title: Define o título do elemento
  • alt: Define um texto alternativo e, por isso, é muito utilizado em imagens, para descrição da mesma
  • hidden: Oculta o elemento
  • align: Permite definir o padrão de alinhamento desse elemento
  • width: Define uma largura para o elemento
  • height: Deixa o texto em negrito, não é a melhor opção, porque não gera responsividade
  • : Define uma altura para o elemento

Tags de Texto

(voltar)

De h1 até h6 são tags de texto ou tipografia, significa heading (cabeçalho) usadas ao definir o tamanho do texto e quando desejo destacá-lo. O h1 é muito utilizada em títulos, porque apresentar o maior tamanho de fonte, com redução à medida que a numeração aumenta. Dentro delas utilizamos a tag p que representa os parágrafos de um texto. A tag hr cria uma linha separadora abaixo do título da página. Para separarmos em linhas diferentes, o texto ou próprio parágrafos usa-se a tag br.

Tags utilizadas em texto:

  • Tag tipo b: Deixa o texto em negrito, não é a melhor opção, porque não gera responsividade
  • Tag tipo strong: Deixa o texto em negrito e o torna responsivo
  • Tag tipo em: Deixa o texto em enfatizado
  • Tag tipo blockquote:
    Define uma citação em um texto
  • Tag tipo mark: O texto fica marcado
  • Tag tipo small: Deixa o texto menor
  • Tag tipo del: O texto é excluído
  • Tag tipo ins: Define um texto que é inserido
  • Tag tipo sub: O texto é subscrito
  • Tag tipo sup: O texto é sobrescrito

Listas

(voltar)

As listas são representadas de duas formas, uma como lista ordenada(ol), ou uma lista não ordenada (ul), ambas com tags de abertura e fechamento. Os elementos são representados dentro da tag lista pela tag li, que forma os itens da lista.

Segue exemplo de listas ordenada e não ordenada respectivamente:

Lista Ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Lista não Ordenada:

  • Item 1
  • Item 2
  • Item 3

Links

(voltar)

A tag de link do HTML é responsável por fazer a ligação entre um documento e outro, podendo ser um link da mesma página ou de uma página de outro domínio.Para fazer um link, usa-se a tag (a, chamada de âncora) com o atributo href (que contém a url da página).
